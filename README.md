A real-time strategy game for Windows developed in Visual Basic 6 in 2006.

## Features

- Real-time strategy gameplay
- Built with Visual Basic 6
- Custom pre-rendered 3D graphics created in 3D Studio Max
- Custom-designed icons and UI
- Original campaign (~40 minutes of gameplay with multiple objectives)
- Music and sound effects from Worms World Party
  
## About

This was a personal project built when I was 15 years old.
It started as a small RTS experiment and slowly grew beyond what I could maintain with VB6 tooling at the time.
It was eventually abandoned, but kept in a playable state.

## Running the game

- Set your screen resolution to **1024×768** for the intended experience.
- Customization instructions are available inside each asset folder.

Enjoy!
