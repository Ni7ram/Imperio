IMPERIO

- RTS for Windows made in Visual Basic 6
- Custom 3d pre-rendered graphics made with 3d Studio Max
- Custom icons
- Music & SFX taken from Worms World Party

All made by Martin Cardozo at 15 y.o. 

This project was abandoned due to ever increasing complexity, but it was left in a playable state (40 min of game aprox. with different objectives).

Customization instructions inside every asset folder.

Important: set your screen resolution at 1024x768 for a correct experience.

Have fun!
